# WebAR推理ゲーム

モバイルのWebブラウザで遊べるAR推理ゲームです。

## 使用技術
 - フロントエンド
   - Next.js
   - TypeScript
 - バックエンド
   - Supabase
 - AR
   - AR.js
   - THREE.js

## ARマーカー
サイトにアクセスし、サイト上のカメラで以下のマーカーを映すと、手がかりの3Dモデルが表示されます。 <br />
これらのマーカーを探し、物語の情報を得て、推理を進めていくゲームです。
